# Yoni Elentok Dotfiles
- Linux Distribution: ubuntu 20.04
- Window Manager: i3
- Test Editor: neovim
- IDE: neovim
- Git interface: tig
- File manager: Ranger
- Browser: Firefox

## Installation

```
sudo apt-get install git -y
git clone https://github.com/elyoni/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
